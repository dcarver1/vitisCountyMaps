## Project Structure

```
vitisCountyMaps/
├── data/
│   ├── raw/          # Raw input data (species occurrence records, shapefiles, etc.)
│   └── processed/    # Cleaned and processed data ready for analysis
├── R/
│   ├── functions/    # Reusable R functions
│   └── *.R           # Main analysis and map generation scripts
├── output/
│   └── maps/         # Generated county map outputs
└── docs/             # Project documentation
```
